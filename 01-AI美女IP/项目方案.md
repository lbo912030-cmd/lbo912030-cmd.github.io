# 项目一：AI美女IP方案

## 赛道概述

用AI工具生成美女图片，制作短视频，在国内外平台发布并变现。流量获取能力强，但合规风险是所有赛道中最高的。

## 变现路径

### 海外路线（主力）
| 平台 | 模式 | 收益参考 |
|------|------|----------|
| Fanvue | AI虚拟创作者订阅 | 头部月入2万美元+ |
| OnlyFans | 2026年已允许AI内容（需标注） | 日本案例：AI美女月入$13,450 |
| TikTok/Instagram/Reddit | 导流获客 | 7天120万播放、4678粉丝案例 |

操作流程：AI生成图片 → 社交平台发图导流 → 引流至订阅平台 → 付费订阅（$9.9/月）→ ChatGPT自动回复私信

### 国内路线（受限）
- 品牌商单（需高质量内容）
- 收徒卖课
- 平台流量分成（收益低）

## 平台政策红线（国内）

| 平台 | 处罚力度 |
|------|----------|
| 抖音 | 2026年1-3月处置4.2万条AI低俗内容，处罚1.4万个账号；擦边直接永久封禁；涉刑已有多人被拘留 |
| 小红书 | 封禁超80万个AI托管账号；AI内容未标注直接限流下架 |
| 微信 | 严打AI生成低俗内容 |

**必须遵守**：
- AI生成内容强制标注（不标注限流50%-80%）
- 严禁低俗擦边
- 严禁AI全托管运营

## 所需工具

| 环节 | 工具 |
|------|------|
| 图片生成 | Midjourney、Stable Diffusion |
| 视频生成 | 可灵、Runway、Seedance |
| 文字/人设 | Claude、ChatGPT |
| 海外分发 | TikTok、Instagram、Reddit |
| 订阅变现 | Fanvue、OnlyFans |

## 风险评估

- 监管持续加码，国内擦边路线基本走不通
- 海外路线需要稳定的网络、账号体系、收款渠道、英语运营能力
- 赛道已从"技术红利期"进入"内容深耕期"，同质化严重
- 2025年9月《人工智能生成合成内容标识办法》已正式施行

## 建议

若要做，走**海外正规路线**。国内擦边坚决不碰。需具备海外运营全套能力，否则很难起步。
